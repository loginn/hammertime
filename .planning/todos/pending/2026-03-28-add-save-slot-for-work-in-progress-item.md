## Problem

Currently the single crafting bench is the only place to hold an item being worked on. If a new base drops, you have to finish or discard your current project. A dedicated "work in progress" save slot would let players park an in-progress craft.

## Solution

Add a WIP save slot alongside the crafting bench — player can stash one item mid-craft and pick it up later. Separate from the 3-slot stash (which holds unworked bases).
