**3. Update get_rarity_weights()** to find the largest threshold <= area_level instead of using `min(area_level, 5)`:
```gdscript
static func get_rarity_weights(area_level: int) -> Dictionary:
    var lookup_level = 1
    for threshold in [500, 300, 200, 100, 1]:
        if area_level >= threshold:
            lookup_level = threshold
            break
    return RARITY_WEIGHTS[lookup_level]
```

**4. Add _calculate_currency_chance() static helper** (before roll_currency_drops):
```gdscript
static func _calculate_currency_chance(
    base_chance: float,
    area_level: int,
    unlock_level: int,
    ramp_duration: int = 50
) -> float:
    if area_level < unlock_level:
        return 0.0
    var levels_since_unlock = area_level - unlock_level
    if levels_since_unlock >= ramp_duration:
        return base_chance
    var ramp_progress = float(levels_since_unlock) / float(ramp_duration)
    var ramp_multiplier = 0.1 + (0.9 * ramp_progress)
    return base_chance * ramp_multiplier
```
Start at 10% of base chance at unlock, linearly ramp to 100% over 50 area levels. This means Forge (unlocks at 100) reaches full strength at area 150, Grand at 250, Claw/Tuning at 350.

**5. Modify roll_currency_drops()** to add hard gating and ramping:
- At the top of the for-loop over currency_rules, add hard gate check:
  ```gdscript

  if area_level < unlock_level:
      continue
  ```
- After the gate check, apply ramping for non-starter currencies (unlock_level > 1):
  ```gdscript
  var adjusted_chance = rule["chance"]
  if unlock_level > 1:
      adjusted_chance = _calculate_currency_chance(rule["chance"], area_level, unlock_level, 50)
  if randf() < adjusted_chance:
      # ... existing drop logic
  ```
- Update bonus drop distribution to be uniform: change existing line `var random_currency = dropped_currencies[randi() % dropped_currencies.size()]` — this already distributes uniformly across currencies that dropped. However, ensure we use ALL currencies that passed the gate (not just those that dropped) by building an `eligible_currencies` list from currencies that passed the area gate, then distributing bonus drops to eligible currencies regardless of whether they already dropped:
  ```gdscript
  var bonus_drops = area_level - 1
  if bonus_drops > 0:
      var eligible_currencies = []
      for currency_name_check in currency_rules:
          if area_level >= CURRENCY_AREA_GATES[currency_name_check]:
              eligible_currencies.append(currency_name_check)
      for i in range(bonus_drops):
          var random_currency = eligible_currencies[randi() % eligible_currencies.size()]
          if random_currency in drops:
              drops[random_currency] += 1
          else:
              drops[random_currency] = 1
  ```
  This gives advanced currencies a chance to benefit from bonus drops even if their initial roll failed, preventing Runic/Tack from absorbing all bonus drops at high areas. This addresses the "Pitfall 3" from research.
