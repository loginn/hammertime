  <action>
    **SaveManager (autoloads/save_manager.gd):**

    Add `export_save_string() -> String` method:
    1. Call `_build_save_data()` to get full game state dictionary
    2. `JSON.stringify(save_data)` — compact, no indent (minimize string length)
    3. `Marshalls.utf8_to_base64(json_string)` to encode
    4. Compute checksum: `base64.md5_text()` — 32-char hex string
    5. Return `"HT1:" + base64 + ":" + checksum`

    Add `import_save_string(save_string: String) -> Dictionary` method returning `{"success": bool, "error": String}`:
    1. Strip whitespace/newlines from input: `save_string.strip_edges().replace("\n", "").replace("\r", "").replace(" ", "")`
    2. Check prefix: `if not cleaned.begins_with("HT1:")` → return `{"success": false, "error": "not_hammertime"}`
    3. Strip "HT1:" prefix (substr(4)), find last ":" with rfind to split base64 and checksum
    4. If no colon found → return `{"success": false, "error": "invalid_format"}`
    5. Verify checksum: `base64_part.md5_text() != checksum_part` → return `{"success": false, "error": "corrupted"}`
    6. Decode: `Marshalls.base64_to_utf8(base64_part)` — if empty → return `{"success": false, "error": "decode_failed"}`
    7. Parse JSON: `JSON.parse_string(json_string)` — if null or not Dictionary → return `{"success": false, "error": "invalid_json"}`
    8. Version check: if `int(data.get("version", 0)) > SAVE_VERSION` → return `{"success": false, "error": "newer_version"}`
    9. Call `_migrate_save(data)` then `_restore_state(data)`
    10. If restore fails → return `{"success": false, "error": "restore_failed"}`
    11. Call `save_game()` to persist to disk immediately
    12. Return `{"success": true, "error": ""}`

    **GameEvents (autoloads/game_events.gd):**

    Add signal: `signal import_completed()` — emitted after successful import so views can refresh.

    After successful import in SaveManager.import_save_string, emit `GameEvents.import_completed.emit()`.

    - Tween fades alpha as before: `tween_property(self, "modulate:a", 0.0, 0.5)`

    - Error color: `Color(1.0, 0.4, 0.4)` — soft red, readable on dark background
    - Success color: `Color(0.4, 1.0, 0.4)` — soft green for import/export success
