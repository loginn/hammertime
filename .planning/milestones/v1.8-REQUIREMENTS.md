# Requirements: v1.8 Content Pass — Items & Mods

## v1.8 Requirements

### Item Bases (BASE)

- [x] **BASE-01**: Rename existing items to thematic names (BasicArmor -> Iron Plate, BasicHelmet -> Iron Helm, BasicBoots -> Iron Greaves, BasicRing -> Iron Band, LightSword -> Broadsword or similar)
- [x] **BASE-02**: 3 STR weapons with varied implicits (e.g., Broadsword/attack speed, Battleaxe/physical damage, Warhammer/bleed chance)
- [x] **BASE-03**: 3 DEX weapons with varied implicits (e.g., Dagger/crit chance, Venom Blade/poison damage, Shortbow/attack speed)
- [x] **BASE-04**: 3 INT weapons with varied implicits (e.g., Wand/spell damage, Lightning Rod/lightning damage, Sceptre/burn damage)
- [x] **BASE-05**: 3 armor bases (Iron Plate/armor, Leather Vest/evasion, Silk Robe/energy shield)
- [x] **BASE-06**: 3 helmet bases (Iron Helm/armor, Leather Hood/evasion, Circlet/energy shield)
- [x] **BASE-07**: 3 boots bases (Iron Greaves/armor, Leather Boots/evasion, Silk Slippers/energy shield)
- [x] **BASE-08**: 3 ring bases (Iron Band/attack damage, Jade Ring/crit, Sapphire Ring/spell damage)
- [x] **BASE-09**: Each base has archetype-appropriate valid_tags constraining affix pool (STR=PHYSICAL/ATTACK/ARMOR, DEX=PHYSICAL/ATTACK/CRITICAL/EVASION, INT=SPELL/ELEMENTAL/ENERGY_SHIELD)
- [x] **BASE-10**: Item serialization registry updated (create_from_dict match arms for all 21 types)

### Spell Damage Channel (SPELL)

- [x] **SPELL-01**: 3 new StatTypes added to tag.gd (FLAT_SPELL_DAMAGE, INCREASED_SPELL_DAMAGE, INCREASED_CAST_SPEED)
- [x] **SPELL-02**: SPELL tag constant added to tag.gd alongside existing ATTACK tag
- [x] **SPELL-03**: Weapon base class gains optional spell damage fields (base_spell_damage_min/max, base_cast_speed) defaulting to 0
- [x] **SPELL-04**: StatCalculator gets calculate_spell_damage_range() and calculate_spell_dps() static methods mirroring attack equivalents
- [x] **SPELL-05**: Hero tracks spell_damage_ranges dict and total_spell_dps parallel to attack damage stats
- [x] **SPELL-06**: CombatEngine adds hero_spell_timer as third independent timer, only starts when equipped weapon has cast_speed > 0
- [x] **SPELL-07**: Hero View shows Attack DPS and Spell DPS separately, hiding whichever channel is 0

### Damage Over Time (DOT)

- [x] **DOT-01**: New StatTypes for DoT added to tag.gd (BLEED_DAMAGE, POISON_DAMAGE, BURN_DAMAGE)
- [x] **DOT-02**: DoT tick system in CombatEngine — debuffs applied on hit, deal damage over configurable duration
- [x] **DOT-03**: Bleed affix (physical DoT, STR signature — uses Tag.PHYSICAL)
- [x] **DOT-04**: Poison affix (DEX signature)
- [x] **DOT-05**: Burn affix (fire DoT, INT signature — uses Tag.FIRE)
- [x] **DOT-06**: DoT damage shown in combat UI (ticking numbers or debuff indicator)
- [x] **DOT-07**: DoT defense interaction design (e.g., burn bypasses armor, bleed bypasses ES, or simpler flat application)

### Affix Pool Expansion (AFF)

- [x] **AFF-01**: Flat Spell Damage prefix added to item_affixes.gd (tags: SPELL, FLAT, WEAPON; stat: FLAT_SPELL_DAMAGE)
- [x] **AFF-02**: %Spell Damage prefix added to item_affixes.gd (tags: SPELL, PERCENTAGE, WEAPON; stat: INCREASED_SPELL_DAMAGE)
- [x] **AFF-03**: Cast Speed suffix enabled with INCREASED_CAST_SPEED stat type (currently disabled stub)
- [x] **AFF-04**: Evade suffix enabled with FLAT_EVASION stat type (currently disabled stub)
- [x] **AFF-05**: Bleed, Poison, and Burn damage suffixes added/enabled with DoT stat types
- [x] **AFF-06**: STR/DEX/INT metadata tag constants added to tag.gd for archetype identity on item bases

### Inventory Rework (INV)

- [x] **INV-01**: 5 crafting benches (one per equipment slot), each holds max 1 item
- [x] **INV-02**: Player selects which bench to craft on via slot tabs (existing UI pattern)
- [x] **INV-03**: Drops for a slot are discarded if bench already occupied — player must craft/equip/discard to open the slot
- [x] **INV-04**: ForgeView shows 5 bench slots instead of 10-item arrays per slot
- [x] **INV-05**: Save format simplified (1 item per slot instead of inventory arrays)

### Loot & Integration (LOOT)

- [x] **LOOT-01**: All 21 bases in drop pool with slot-first-then-archetype random distribution
- [x] **LOOT-02**: Bump save version, delete incompatible old saves on load (no migration)
- [x] **LOOT-03**: Combined DPS (attack + spell + DoT) used for item comparison
- [x] **LOOT-04**: Archetype label visible on items in inventory and crafting views

## Future Requirements

- [ ] Loot filter system for archetype filtering (FILT-01 — target P3+ prestige unlock)
- [ ] Dodge Chance suffix — needs design decision (additive to evasion or separate roll)
- [ ] Damage Suppression suffix — needs DefenseCalculator integration
- [ ] Hybrid bases (STR/DEX, STR/INT, DEX/INT split defense)
- [ ] Mana as spell resource
- [ ] ES identity buff for spellcasters (deferred to hero milestone)

## Out of Scope

- Attribute requirements (STR/DEX/INT as hero stats) — items carry archetype identity, no stat gating
- Class system — archetypes emerge from item choice, not class selection
- Dual-wielding — single weapon slot
- Mana costs for spells — auto-cast on timer, idle-friendly
- Spell skill selection — spells fire automatically
- Save migration — bump version, wipe incompatible saves
- P2-P7 prestige cost tuning — separate balance milestone
