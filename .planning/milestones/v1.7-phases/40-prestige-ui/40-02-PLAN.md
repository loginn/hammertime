# Phase 40, Plan 02: Tab Integration + Fade Transition + Reload

<objective>
Add the prestige tab button ("P{N}" badge) to the tab bar with dynamic visibility gating, integrate prestige_view into the 4-tab system, and implement the fade-to-black transition with scene reload on prestige trigger.
</objective>

<context>

- scenes/main.tscn: Tab bar layout (ForgeTab x=10-130, CombatTab x=140-260, SettingsTab x=1150-1270), ContentArea at y=50, OverlayLayer at layer 10
- scenes/main_view.gd: show_view() match block with 3 cases, tab button disabled state management, settings_view.new_game_started signal connection pattern, _on_new_game_started() calls reload_current_scene()
- scenes/prestige_view.gd (from 40-01): emits prestige_triggered signal, has reset_state()

- autoloads/game_state.gd: prestige_level field
- autoloads/game_events.gd: currency_dropped signal, prestige_completed signal

## User Decisions (from CONTEXT.md)
- Tab bar "P3" badge that IS the prestige tab button (one element, two purposes)
- Hidden at P0, appears when first prestige is affordable, permanent once shown
- Dedicated Prestige tab (4th tab alongside Forge/Combat/Settings)
- Fade to black 0.5s then scene reload via get_tree().reload_current_scene()
- FadeRect lives in OverlayLayer in main.tscn (covers tab bar + all views)
</context>

**Add _check_prestige_tab_reveal() function:**
This handles initial check (from _ready) and currency_dropped signal (which passes a Dictionary). Use a default parameter to handle both call patterns:
```gdscript
func _check_prestige_tab_reveal(_drops: Dictionary = {}) -> void:
    if prestige_tab_revealed:
        return
    if GameState.prestige_level > 0 or PrestigeManager.can_prestige():
        prestige_tab_revealed = true
        prestige_tab.visible = true
        prestige_tab.text = "P" + str(GameState.prestige_level)
```

**Add _on_prestige_triggered() function:**
Handles the fade-to-black transition and scene reload. The prestige_view already called execute_prestige() and save_game() before emitting.
```gdscript
func _on_prestige_triggered() -> void:
    # Show fade rect and block input during fade
    fade_rect.visible = true
    fade_rect.color = Color(0, 0, 0, 0)
    fade_rect.mouse_filter = 0  # STOP — block all input during fade

    # Tween fade to opaque black over 0.5s
    var tween = create_tween()
    tween.tween_property(fade_rect, "color:a", 1.0, 0.5)
    tween.tween_callback(_do_prestige_reload)


func _do_prestige_reload() -> void:
    get_tree().reload_current_scene()
```
