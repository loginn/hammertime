Create a new minimal scene: floating_label.tscn with a Label root node and floating_label.gd script attached.

floating_label.gd should:

1. Extend Label.

2. Provide show_damage(value: int, is_crit: bool) -> void:
   - Set text = str(value)
   - If is_crit:
     - Set font color to gold: add_theme_color_override("font_color", Color(1.0, 0.8, 0.0))
     - Set scale = Vector2(1.5, 1.5) for bigger numbers
     - Set pivot_offset to center of text so scale doesn't shift position
   - Else (normal hit):
     - Set font color to white: add_theme_color_override("font_color", Color(1.0, 1.0, 1.0))
     - scale = Vector2(1.0, 1.0)
   - Create tween: var tween = create_tween()
   - tween.set_parallel(true)
   - Animate position:y upward by 50-60 pixels over 1.0 seconds
   - Animate modulate:a from 1.0 to 0.0 over 1.0 seconds
   - await tween.finished then queue_free()

3. Provide show_dodge() -> void:
   - Set text = "DODGE"
   - Set font color to white: Color(1.0, 1.0, 1.0)
   - Slightly smaller scale or same as normal
   - Create tween with parallel position:y drift up ~40px over 0.8s and modulate:a fade over 0.8s
   - await tween.finished then queue_free()

4. Do NOT color-code by element type (per user decision: "Damage numbers are uniform color regardless of element type").

floating_label.tscn:
- Root node: Label (type Label)
- Script: floating_label.gd
- Set a reasonable default font size (e.g., 20px via theme override) so numbers are readable
- Horizontal alignment center so numbers look centered at spawn point
