<task type="auto">
  <name>Task 2: Create CombatEngine with state machine and dual attack timers</name>
  <files>models/combat/combat_engine.gd</files>
  <action>
Create `models/combat/combat_engine.gd` as a Node (needs child Timers). Create the `models/combat/` directory first.

**Class structure:**
```gdscript
class_name CombatEngine extends Node
```

**State machine enum:**
```gdscript
enum State { IDLE, FIGHTING, MAP_COMPLETE, HERO_DEAD }
```

**Properties:**
- `state: State = State.IDLE`
- `current_packs: Array[MonsterPack] = []`
- `current_pack_index: int = 0`
- `area_level: int = 1`
- `max_unlocked_level: int = 1`
- `auto_retry: bool = true` (default on per Claude's discretion)
- `hero_attack_speed: float = 1.0` (cached from weapon)

**Child Timers (created in _ready):**
- `hero_attack_timer: Timer` -- one_shot = false (repeating)
- `pack_attack_timer: Timer` -- one_shot = false (repeating)

In `_ready()`, create both Timers programmatically (no .tscn needed):
```gdscript
func _ready() -> void:
    hero_attack_timer = Timer.new()
    hero_attack_timer.one_shot = false
    add_child(hero_attack_timer)
    hero_attack_timer.timeout.connect(_on_hero_attack)

    pack_attack_timer = Timer.new()
    pack_attack_timer.one_shot = false
    add_child(pack_attack_timer)
    pack_attack_timer.timeout.connect(_on_pack_attack)
```

**Core methods:**

`start_combat(level: int) -> void`:
- Set `area_level = level`
- Generate packs: `current_packs = PackGenerator.generate_packs(area_level)`
- Reset `current_pack_index = 0`
- Set `state = State.FIGHTING`
- Emit `GameEvents.combat_started.emit(area_level, current_packs.size())`
- Call `_start_pack_fight()`

`stop_combat() -> void`:
- Stop both timers
- Set `state = State.IDLE`
- Emit `GameEvents.combat_stopped.emit()`

`_start_pack_fight() -> void`:
- Cache hero attack speed: if weapon equipped, use `weapon.base_attack_speed`, else default 1.0
- `hero_attack_timer.wait_time = 1.0 / hero_attack_speed`
- `pack_attack_timer.wait_time = 1.0 / get_current_pack().attack_speed`
- Start both timers

`_on_hero_attack() -> void`:
- Guard: if state != FIGHTING, return
- Get current pack, guard null/dead
- Calculate damage per hit: `GameState.hero.total_dps / hero_attack_speed`
  - This removes the speed factor from DPS, giving raw damage per swing
- Roll crit: `var is_crit := randf() < (GameState.hero.total_crit_chance / 100.0)`
- If crit: multiply damage by `GameState.hero.total_crit_damage / 100.0`
- `pack.take_damage(damage)`
- Emit `GameEvents.hero_attacked.emit(damage, is_crit)`
- If `not pack.is_alive()`: call `_on_pack_killed()`

`_on_pack_attack() -> void`:
- Guard: if state != FIGHTING, return
- Get current pack, get hero
- Call `DefenseCalculator.calculate_damage_taken()` with:
  - `pack.damage` as raw_damage
  - `pack.element` as damage_type
  - `false` for is_spell (all pack attacks are attacks, not spells)
  - All hero defense stats from `GameState.hero.get_total_*()` methods
  - `GameState.hero.get_current_energy_shield()` for current ES
- Emit `GameEvents.pack_attacked.emit(result)`
- If `result["dodged"]`: return (dodged, no damage)
- `GameState.hero.apply_damage(result["life_damage"], result["es_damage"])`
- If `not GameState.hero.is_healthy()`: call `_on_hero_died()`

`_on_pack_killed() -> void`:
- Stop both timers
- Increment `current_pack_index`
- Emit `GameEvents.pack_killed.emit(current_pack_index, current_packs.size())`
- If `current_pack_index >= current_packs.size()`: call `_on_map_completed()`, return
- ES recharge between packs (33%): `GameState.hero.recharge_energy_shield()`
- Instant transition: call `_start_pack_fight()` immediately (no delay)

`_on_map_completed() -> void`:
- `state = State.MAP_COMPLETE`
- Full ES recharge: `GameState.hero.current_energy_shield = float(GameState.hero.total_energy_shield)`
- `area_level += 1` (deterministic progression, NOT random)
- `max_unlocked_level = max(max_unlocked_level, area_level)`
- Emit `GameEvents.map_completed.emit(area_level - 1)` (emit the level that was completed)
- Auto-advance: call `start_combat(area_level)` immediately

`_on_hero_died() -> void`:
- `state = State.HERO_DEAD`
- Stop both timers
- `GameState.hero.revive()` (restores full HP + full ES)
- Emit `GameEvents.hero_died.emit()`
- Hero stays at SAME area level (no regression)
- If `auto_retry`: call `start_combat(area_level)` (packs re-roll fresh)
- If NOT auto_retry: remain in HERO_DEAD state (UI will need to call start_combat to resume)

`get_current_pack() -> MonsterPack`:
- Guard: if `current_pack_index >= current_packs.size()`, return null
- Return `current_packs[current_pack_index]`

**Helper for hero attack speed:**
```gdscript
func _get_hero_attack_speed() -> float:
    var weapon = GameState.hero.equipped_items.get("weapon")
    if weapon != null and weapon is Weapon:
        return weapon.base_attack_speed
    return 1.0  # Default unarmed speed
```
Cache this in `hero_attack_speed` at the start of each pack fight.

**IMPORTANT implementation notes:**
- Do NOT use `GameState.hero.total_dps` directly as damage. DPS already includes attack speed. Divide by attack speed to get per-hit damage.
- The `_on_pack_attack` damage comes from `pack.damage` (which is already scaled by PackGenerator). This is damage PER HIT at the pack's attack speed.
- Both timers are repeating (one_shot = false) so they fire continuously until stopped.
- Stop BOTH timers on every state transition exit (pack killed, hero died, map complete, stop_combat).
