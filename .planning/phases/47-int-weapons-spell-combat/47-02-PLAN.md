# Plan 02: CombatEngine Spell Timer & Combat Feedback

## Objective
Add the `is_spell_user` boolean to Hero, wire the CombatEngine to use a spell timer (instead of attack timer) when the hero is a spell user, add the `hero_spell_hit` signal for distinct combat feedback (purple floating text), persist `is_spell_user` in the save system, and add a temporary dev toggle in settings view.

## must_haves
- [ ] Hero has `is_spell_user: bool = false` property
- [ ] CombatEngine creates a third timer (`hero_spell_timer`) in `_ready()` and uses it instead of `hero_attack_timer` when `hero.is_spell_user` is true
- [ ] `_on_hero_spell_hit()` rolls damage from `hero.spell_damage_ranges` (all elements), applies shared crit, calls `pack.take_damage()`, and emits `GameEvents.hero_spell_hit`
- [ ] `_get_hero_cast_speed()` reads cast speed from equipped weapon with fallback 1.0
- [ ] `_stop_timers()` stops all three timers
- [ ] `GameEvents.hero_spell_hit` signal exists and gameplay_view connects it for purple floating text
- [ ] `is_spell_user` is saved/loaded via SaveManager
- [ ] Settings view has a temporary "Spell Mode" toggle that sets `is_spell_user`

### Task 3: Wire CombatEngine spell timer

**Do:**
- In `models/combat/combat_engine.gd`:
  1. Add `var hero_spell_timer: Timer` declaration alongside existing timer vars (line 18-19 area).
  2. In `_ready()`, create and add the spell timer (same pattern as hero_attack_timer):
     ```
     hero_spell_timer = Timer.new()
     hero_spell_timer.one_shot = false
     add_child(hero_spell_timer)
     hero_spell_timer.timeout.connect(_on_hero_spell_hit)
     ```
  3. Modify `_start_pack_fight()` to branch on `GameState.hero.is_spell_user`:
     ```
     if GameState.hero.is_spell_user:
         hero_spell_timer.wait_time = 1.0 / _get_hero_cast_speed()
         hero_spell_timer.start()
     else:
         hero_attack_speed = _get_hero_attack_speed()
         hero_attack_timer.wait_time = 1.0 / hero_attack_speed
         hero_attack_timer.start()
     pack_attack_timer.wait_time = 1.0 / get_current_pack().attack_speed
     pack_attack_timer.start()
     ```
  4. Add `_on_hero_spell_hit()` method mirroring `_on_hero_attack()`:
     ```
     func _on_hero_spell_hit() -> void:
         if state != State.FIGHTING:
             return
         var pack := get_current_pack()
         if pack == null or not pack.is_alive():
             return
         var hero := GameState.hero
         var damage_per_hit := 0.0
         for element in hero.spell_damage_ranges:
             var el_min: float = hero.spell_damage_ranges[element]["min"]
             var el_max: float = hero.spell_damage_ranges[element]["max"]
             if el_max > 0.0:
                 damage_per_hit += randf_range(el_min, el_max)
         var is_crit := randf() < (hero.total_crit_chance / 100.0)
         if is_crit:
             damage_per_hit *= (hero.total_crit_damage / 100.0)
         pack.take_damage(damage_per_hit)
         GameEvents.hero_spell_hit.emit(damage_per_hit, is_crit)
         if not pack.is_alive():
             _on_pack_killed()
     ```
  5. Add `_get_hero_cast_speed()` method:
     ```
     func _get_hero_cast_speed() -> float:
         var weapon = GameState.hero.equipped_items.get("weapon")
         if weapon != null and weapon is Weapon:
             if weapon.base_cast_speed > 0.0:
                 return weapon.base_cast_speed
         return 1.0
     ```
  6. Update `_stop_timers()` to also stop the spell timer:
     ```
     func _stop_timers() -> void:
         hero_attack_timer.stop()
         hero_spell_timer.stop()
         pack_attack_timer.stop()
     ```
