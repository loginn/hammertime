<task id="46-03-01">
<title>Add test group 22: Spell damage data model and StatCalculator</title>
<description>
Add `_group_22_spell_damage_data_model()` to tools/test/integration_test.gd and call it from _ready().

Tests:
1. **Weapon spell field defaults:** Construct Broadsword(8), verify base_spell_damage_min == 0, base_spell_damage_max == 0, base_cast_speed == 0.0, spell_dps == 0.0
2. **Ring cast speed defaults:** Construct IronBand(8), verify base_cast_speed == 0.0. Construct JadeRing(8), verify base_cast_speed == 0.0.
3. **SapphireRing cast speed:** Construct SapphireRing(8), verify base_cast_speed == 0.5. Construct SapphireRing(1), verify base_cast_speed == 1.2.
4. **calculate_spell_damage_range basic:** Call StatCalculator.calculate_spell_damage_range(10, 20, []), verify result has "spell" key with min=10.0, max=20.0
5. **calculate_spell_damage_range with flat affix:** Create a FLAT_SPELL_DAMAGE affix (add_min=5, add_max=10), pass to calculate_spell_damage_range(10, 20, [affix]), verify spell.min == 15.0, spell.max == 30.0
6. **calculate_spell_dps zero cast speed:** Call calculate_spell_dps(15.0, 0.0, []), verify result == 0.0
7. **calculate_spell_dps basic:** Call calculate_spell_dps(15.0, 1.0, []), verify result approximately == 15.375 (15 * 1.0 * 1.025 crit)
8. **calculate_spell_dps with %spell damage:** Create INCREASED_SPELL_DAMAGE affix (value=100), call calculate_spell_dps(10.0, 1.0, [affix]), verify damage is doubled before speed/crit
</description>

<title>Add test group 23: Hero spell stat tracking</title>
<description>
Add `_group_23_hero_spell_stats()` to tools/test/integration_test.gd and call it from _ready().

7. **Spell damage ranges populated:** Equip SapphireRing(8), verify spell_damage_ranges["spell"]["min"] > 0 or spell_damage_ranges["spell"]["max"] > 0

<title>Add test group 24: Serialization of spell fields</title>
<description>
Add `_group_24_spell_field_serialization()` to tools/test/integration_test.gd and call it from _ready().

Tests:
1. **Weapon round-trip preserves spell fields:** Create Broadsword(8), set base_spell_damage_min = 5, base_spell_damage_max = 15, base_cast_speed = 1.0. Call to_dict(), then create_from_dict(), verify spell fields preserved.
   - Note: If weapon spell fields are computed from base class (not saved/loaded separately), this test verifies that the default 0 values survive round-trip correctly. Adjust based on whether spell fields need explicit serialization.
2. **SapphireRing round-trip:** Create SapphireRing(8), call to_dict(), create_from_dict(), verify base_cast_speed == 0.5 and spell_dps matches original.
3. **IronBand round-trip:** Create IronBand(8), call to_dict(), create_from_dict(), verify base_cast_speed == 0.0.

Note: base_spell_damage and base_cast_speed on weapons default to 0 and are set by subclass _init(). They likely don't need explicit serialization since create_from_dict() calls the subclass constructor which sets them. But this test verifies the round-trip is correct regardless.
