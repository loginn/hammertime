**2. Affix display format for flat damage:**
Add a helper function to forge_view.gd that formats an affix line, distinguishing flat damage affixes:

```gdscript
func _format_affix_line(affix: Affix) -> String:
	if Tag.StatType.FLAT_DAMAGE in affix.stat_types and (affix.add_min > 0 or affix.add_max > 0):
		var element_name := _get_affix_element_name(affix.tags)
		return "Adds %d to %d %s Damage" % [affix.add_min, affix.add_max, element_name]
	return affix.affix_name + ": " + str(affix.value)


func _get_affix_element_name(tags: Array) -> String:
	if Tag.FIRE in tags:
		return "Fire"
	if Tag.COLD in tags:
		return "Cold"
	if Tag.LIGHTNING in tags:
		return "Lightning"
	return "Physical"
```

**3. Update all affix display loops in get_item_stats_text():**
In every section where prefixes and suffixes are displayed (Weapon, Ring, Armor, Boots, Helmet), replace the format line:

This changes ALL prefix/suffix display lines across all 5 item types (weapon, ring, armor, boots, helmet). Non-flat-damage affixes will still show "Name: value" because the helper falls through to the else branch.
