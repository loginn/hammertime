  <files>models/currencies/tack_hammer.gd, models/currencies/grand_hammer.gd</files>
  <action>
Create two Currency subclasses that add mods to items of specific rarities.

**TackHammer** (models/currencies/tack_hammer.gd):
- `class_name TackHammer extends Currency`
- `_init()`: set currency_name = "Tack Hammer"
- `can_apply(item: Item) -> bool`: returns true only if:
  1. item.rarity == Item.Rarity.MAGIC AND
  2. Item has room for at least one more mod: `item.prefixes.size() < item.max_prefixes() or item.suffixes.size() < item.max_suffixes()`
- `get_error_message(item: Item) -> String`:
  - If item.rarity != MAGIC: "Tack Hammer can only be used on Magic items"
  - If at mod limit: "Item already has maximum mods for Magic rarity"
- `_do_apply(item: Item) -> void`:
  1. Try to add one random mod. Check which slots have room (prefix vs suffix).
  2. If both have room, pick randomly (50/50). If only one has room, use that one.
  3. Call item.add_prefix() or item.add_suffix() accordingly.
  4. Call item.update_value()

**GrandHammer** (models/currencies/grand_hammer.gd):
- `class_name GrandHammer extends Currency`
- `_init()`: set currency_name = "Grand Hammer"
- `can_apply(item: Item) -> bool`: returns true only if:
  1. item.rarity == Item.Rarity.RARE AND
  2. Item has room for at least one more mod: `item.prefixes.size() < item.max_prefixes() or item.suffixes.size() < item.max_suffixes()`
- `get_error_message(item: Item) -> String`:
  - If item.rarity != RARE: "Grand Hammer can only be used on Rare items"
  - If at mod limit: "Item already has maximum mods for Rare rarity"
- `_do_apply(item: Item) -> void`:
  1. Try to add one random mod. Check which slots have room.
  2. If both have room, pick randomly. If only one has room, use that one.
  3. Call item.add_prefix() or item.add_suffix() accordingly.
  4. Call item.update_value()

<task type="auto">
  <name>Task 2: Create ClawHammer and TuningHammer mod-manipulation currencies</name>
  <files>models/currencies/claw_hammer.gd, models/currencies/tuning_hammer.gd</files>
  <action>
Create two Currency subclasses for removing and rerolling mods.

**ClawHammer** (models/currencies/claw_hammer.gd):
- `class_name ClawHammer extends Currency`
- `_init()`: set currency_name = "Claw Hammer"
- `can_apply(item: Item) -> bool`: returns true only if item has at least one explicit mod (prefix or suffix). Works on any rarity (Magic or Rare -- not Normal since Normal has no mods). Check: `item.prefixes.size() > 0 or item.suffixes.size() > 0`
- `get_error_message(item: Item) -> String`:
  - If no mods: "Item has no mods to remove"
- `_do_apply(item: Item) -> void`:
  1. Collect all mods into a combined list. Build an array of dictionaries like `[{"type": "prefix", "index": 0}, {"type": "suffix", "index": 0}, ...]` representing all current mods.
  2. Pick one at random.
  3. If it's a prefix, call `item.prefixes.remove_at(index)`. If suffix, `item.suffixes.remove_at(index)`.
  4. Do NOT change item.rarity (CRAFT-05 explicitly says "without changing rarity").
  5. Call item.update_value()

**TuningHammer** (models/currencies/tuning_hammer.gd):
- `class_name TuningHammer extends Currency`
- `_init()`: set currency_name = "Tuning Hammer"
- `can_apply(item: Item) -> bool`: returns true only if item has at least one explicit mod. Check: `item.prefixes.size() > 0 or item.suffixes.size() > 0`
- `get_error_message(item: Item) -> String`:
  - If no mods: "Item has no mods to reroll"
- `_do_apply(item: Item) -> void`:
  1. Iterate over all prefixes and call `prefix.reroll()` on each (Affix.reroll() already exists -- it randomizes value between min_value and max_value, which is the tier range).
  2. Iterate over all suffixes and call `suffix.reroll()` on each.
  3. Do NOT reroll the implicit (CRAFT-06 says "rerolls all mod values" -- mods are explicit prefixes/suffixes, not implicits).
  4. Call item.update_value()

All functions must have explicit return type hints.
  </action>
  <verify>Verify both files exist. Check ClawHammer removes a random mod from prefixes or suffixes without changing rarity. Check TuningHammer calls reroll() on all prefixes and suffixes but not the implicit. Both should call update_value().</verify>
  <done>ClawHammer removes random mod preserving rarity (CRAFT-05), TuningHammer rerolls all explicit mod values within tier ranges (CRAFT-06), both validate mod presence before applying.</done>
</task>
