  Replace the entire `compute_dps()` method and simplify `update_value()`:

  ```gdscript
  func update_value() -> void:
      var all_affixes := self.prefixes + self.suffixes
      all_affixes.append(self.implicit)
      self.dps = StatCalculator.calculate_dps(
          self.base_damage, self.base_speed, all_affixes,

      )
  ```

  Keep all existing properties (base_damage, base_speed, dps, phys_dps, bleed_dps, lightning_dps, cold_dps, fire_dps, crit_chance, crit_damage) -- they're part of the Weapon class API. Only the calculation method changes.

  Change `self.dps = self.compute_dps()` to `self.update_value()` in `_init()`. This makes LightSword consistent with all other concrete item classes (BasicRing, BasicArmor, etc.) which already call `self.update_value()` in their _init.
