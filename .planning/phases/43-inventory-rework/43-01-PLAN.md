# Plan 43-01: Single-bench-per-slot inventory rework

## Context
Replace 10-item-per-slot inventory arrays with a single nullable Item per slot. Each of the 5 equipment slots (weapon, helmet, armor, boots, ring) holds max 1 item on its crafting bench. Drops are silently discarded when the bench for that slot is already occupied. Save format bumps to v5 with single item dicts instead of arrays. Melt gets a two-click confirmation matching the existing equip confirmation pattern.

			display_text += "\n"
		else:

	inventory_label.text = display_text
	update_slot_button_labels()
