**Script file (crafting_view.gd):**

Remove all old hammer logic:
- Remove `enum button { NONE, IMPLICIT, PREFIX, SUFFIX }`
- Remove `var button_pressed: button`
- Remove `var hammer_counts: Dictionary`
- Remove `ImplicitHammer_toggled()`, `AddPrefixHammer_toggled()`, `AddSuffixHammer_toggled()`
- Remove the old `update_item()` function body that uses the button enum
- Remove `on_currencies_found()` temporary mapping function
- Remove `@onready var implicit_hammer_btn`, `prefix_hammer_btn`, `suffix_hammer_btn`

Add new currency system:

1. **Currency instances** (class-level vars):
```gdscript
var currencies: Dictionary = {
    "runic": RunicHammer.new(),
    "forge": ForgeHammer.new(),
    "tack": TackHammer.new(),
    "grand": GrandHammer.new(),
    "claw": ClawHammer.new(),
    "tuning": TuningHammer.new()
}
var selected_currency: Currency = null
var selected_currency_type: String = ""
```

2. **@onready references** for 6 new buttons:
```gdscript
@onready var runic_btn: Button = $ButtonControl/RunicHammerBtn
@onready var forge_btn: Button = $ButtonControl/ForgeHammerBtn
@onready var tack_btn: Button = $ButtonControl/TackHammerBtn
@onready var grand_btn: Button = $ButtonControl/GrandHammerBtn
@onready var claw_btn: Button = $ButtonControl/ClawHammerBtn
@onready var tuning_btn: Button = $ButtonControl/TuningHammerBtn
```

3. **Button-to-currency mapping** dictionary:
```gdscript
var currency_buttons: Dictionary = {}
# Initialized in _ready():
# currency_buttons = { "runic": runic_btn, "forge": forge_btn, ... }
```

4. **_ready():** Connect each button's pressed signal to `_on_currency_selected.bind("runic")` etc. Initialize `currency_buttons` dictionary. Remove old button connections.

5. **_on_currency_selected(currency_type: String):**
   - Get the button from `currency_buttons[currency_type]`
   - If button is toggled on: set `selected_currency = currencies[currency_type]` and `selected_currency_type = currency_type`
   - If button is toggled off: set `selected_currency = null` and `selected_currency_type = ""`
   - Untoggle all other currency buttons (call `untoggle_all_other_buttons()` using the existing pattern)

6. **update_item(event: InputEvent):** Replace old button-enum logic with:
   - Guard: if `selected_currency == null`, print "No currency selected" and return
   - Guard: if `current_item == null`, print "No item selected" and return
   - Check `selected_currency.can_apply(current_item)`. If false:
     - Print the error: `selected_currency.get_error_message(current_item)`
     - Return without consuming
   - Check `GameState.spend_currency(selected_currency_type)`. If false:
     - Print "No " + selected_currency.currency_name + " remaining!"
     - Return
   - Call `selected_currency.apply(current_item)` (returns true since we pre-checked can_apply)
   - Print "Applied " + selected_currency.currency_name
   - Call `update_label()`, `update_currency_button_states()`
   - Call `current_item.display()`

7. **update_currency_button_states():** (replaces old `update_hammer_button_states()`)
   - For each currency_type in currency_buttons:
     - Get count from `GameState.currency_counts.get(currency_type, 0)`
     - Get button from `currency_buttons[currency_type]`
     - Set button.disabled = (count <= 0)
     - Set button.text = currencies[currency_type].currency_name + " (" + str(count) + ")"
   - If selected_currency is not null and its count is 0, deselect it

8. **on_currencies_found(drops: Dictionary):** Simplify to just call `update_currency_button_states()` (no mapping needed, counts already in GameState).

9. **finish_item():** Replace `update_hammer_button_states()` with `update_currency_button_states()`. Reset `selected_currency = null` and `selected_currency_type = ""`.

10. In `_ready()`, replace `update_hammer_button_states()` with `update_currency_button_states()`.

All functions must have explicit return type hints (-> void, -> String, etc.).
