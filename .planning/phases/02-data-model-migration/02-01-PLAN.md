<objective>
Convert all data model classes (Affix, Implicit, Item, and all Item subclasses) from extending Node to extending Resource.

Purpose: Resources are Godot's intended base class for pure data objects. Items and affixes have no scene tree presence, no _process loops, no visual components -- they are data containers. Extending Resource gives proper serialization support, reference counting, and removes unnecessary Node overhead. This is the foundational change that DATA-01 and DATA-02 require.

Output: 14 modified GDScript files, all data classes extending Resource instead of Node, with all existing functionality preserved.
</objective>
