# Plan 01: Core DoT Engine

## Context
This plan builds the core DoT runtime: effect tracking on packs and hero, proc logic in CombatEngine hit handlers, tick timer, stacking rules per DoT type, and lifecycle management. All DoT stat types (DOT-01) and affixes already exist in tag.gd and item_affixes.gd — this plan wires the runtime behavior. DOT-03/04/05 are covered here because the affix data already exists; what's missing is the proc + tick logic that makes bleed/poison/burn actually function.

### Task 2: Add DoT effect tracking to MonsterPack

Add DoT effect tracking to `models/monsters/monster_pack.gd`:

```gdscript
# Active DoT effects applied by hero
# Each entry: {"type": String, "damage_per_tick": float, "ticks_remaining": int, "element": String}
var active_dots: Array = []
```

Add helper methods:
- `apply_dot(dot_type: String, damage_per_tick: float, element: String) -> int` — applies a DoT following stacking rules:
  - **Bleed:** Max 8 stacks. If at cap, replace the stack with lowest `ticks_remaining`. Otherwise add new stack. Duration: 4 ticks.
  - **Poison:** Unlimited stacks. Always add a new stack. Duration: 4 ticks.
  - **Burn:** Max 1 stack. Replace existing burn if present. Duration: 4 ticks.
  - Returns the current stack count for this dot_type after application.
- `process_dot_tick() -> Array` — iterates all active_dots, accumulates total damage per type, decrements ticks_remaining, removes expired entries. Returns an array of `{"type": String, "damage": float}` for each type that ticked (used for signals).
- `clear_dots() -> void` — removes all active DoT effects.
- `get_dot_count(dot_type: String) -> int` — returns number of active stacks for a given type.

### Task 4: Add DoT DPS calculation to Hero

Add `calculate_dot_dps()` in `models/hero.gd` (called at end of `calculate_dot_stats()`). This provides the expected DoT DPS for the hero stats display.

**Formulas per type:**

**Bleed DPS** (attack-mode only, `is_spell_user == false`):
- `avg_hit_damage` = average across all attack damage_ranges elements
- `bleed_base_pct` = 0.15 (15% of hit damage per tick, 60% total over 4 ticks)
- `avg_flat_bleed` = `(total_bleed_damage_min + total_bleed_damage_max) / 2.0`
- `bleed_scaling` = `avg_flat_bleed / max(avg_hit_damage, 1.0)` (convert flat to %)
- `tick_damage` = `avg_hit_damage * (bleed_base_pct + bleed_scaling) * (1.0 + total_bleed_damage_pct / 100.0)`
- `effective_stacks` = `min(attack_speed * (total_bleed_chance / 100.0) * 4.0, 8.0)` (capped at 8)
- `bleed_dps` = `tick_damage * effective_stacks` (each stack ticks once/sec)
- If `total_bleed_chance <= 0`, bleed_dps = 0

**Poison DPS** (attack-mode only):
- `avg_flat_poison` = `(total_poison_damage_min + total_poison_damage_max) / 2.0`
- `tick_per_stack` = `avg_flat_poison * (1.0 + total_poison_damage_pct / 100.0)`
- `attack_speed` from weapon
- `effective_stacks` = `attack_speed * (total_poison_chance / 100.0) * 4.0` (unlimited)
- `poison_dps` = `tick_per_stack * effective_stacks`
- If `total_poison_chance <= 0`, poison_dps = 0

**Burn DPS** (spell-mode only, `is_spell_user == true`):
- `avg_spell_hit_damage` = average across all spell_damage_ranges elements
- `burn_base_pct` = 0.25 (25% of hit per tick, 100% total over 4 ticks)
- `avg_flat_burn` = `(total_burn_damage_min + total_burn_damage_max) / 2.0`
- `burn_scaling` = `avg_flat_burn / max(avg_spell_hit_damage, 1.0)`
- `tick_damage` = `avg_spell_hit_damage * (burn_base_pct + burn_scaling) * (1.0 + total_burn_damage_pct / 100.0)`
- `burn_dps` = `tick_damage * (total_burn_chance / 100.0)` (single stack, refreshed)
- If `total_burn_chance <= 0`, burn_dps = 0

`total_dot_dps` = sum of applicable types (bleed+poison if attack-mode, burn if spell-mode).

Add getter: `get_total_dot_dps() -> float`

```gdscript
# Bleed proc (attack-mode only)
if hero.total_bleed_chance > 0.0:
    if randf() * 100.0 < hero.total_bleed_chance:
        var flat_avg := (hero.total_bleed_damage_min + hero.total_bleed_damage_max) / 2.0
        var base_pct := 0.15
        var scaling := flat_avg / maxf(damage_per_hit, 1.0)
        var tick_dmg := damage_per_hit * (base_pct + scaling) * (1.0 + hero.total_bleed_damage_pct / 100.0)
        var count := pack.apply_dot("bleed", tick_dmg, "physical")
        GameEvents.dot_applied.emit("pack", "bleed", count)
        _ensure_dot_timer_running()

# Poison proc (attack-mode only, chance overflow + crit bonus)
if hero.total_poison_chance > 0.0:
    var effective_chance := hero.total_poison_chance
    if is_crit:
        effective_chance += 100.0  # +100% chance on crit
    # Chance overflow: guaranteed stacks + fractional chance for one more
    var guaranteed_stacks := int(effective_chance / 100.0)
    var fractional_chance := fmod(effective_chance, 100.0)
    var total_stacks := guaranteed_stacks
    if randf() * 100.0 < fractional_chance:
        total_stacks += 1
    if total_stacks > 0:
        var flat_avg := (hero.total_poison_damage_min + hero.total_poison_damage_max) / 2.0
        var tick_dmg := flat_avg * (1.0 + hero.total_poison_damage_pct / 100.0)
        var count := 0
        for i in total_stacks:
            count = pack.apply_dot("poison", tick_dmg, "chaos")
        GameEvents.dot_applied.emit("pack", "poison", count)
        _ensure_dot_timer_running()
```

**In `_on_hero_spell_hit()`**, after `pack.take_damage()` and `GameEvents.hero_spell_hit.emit()`, add burn proc:

### Task 6: Add DoT tick processing and pack DoT to CombatEngine

> **Cross-dependency note:** This task calls `DefenseCalculator.calculate_dot_damage_taken()` (Plan 02 Task 1) and `hero.get_total_chaos_resistance()` (Plan 02 Task 2). In single-agent sequential execution, implement Plan 02 before this task, or implement Plan 02 Tasks 1-2 first within the wave.

**Add `_on_dot_tick()` method** to `models/combat/combat_engine.gd`:

```gdscript
func _on_dot_tick() -> void:
    if state != State.FIGHTING:
        dot_tick_timer.stop()
        return

    var pack := get_current_pack()
    var hero := GameState.hero
    var any_dots_active := false

    # Process hero's DoTs on pack
    if pack != null and pack.is_alive():
        # Snapshot active types BEFORE tick to detect expirations
        var pre_tick_types := {}
        for dot_type in ["bleed", "poison", "burn"]:
            var count := pack.get_dot_count(dot_type)
            if count > 0:
                pre_tick_types[dot_type] = true

        var tick_results := pack.process_dot_tick()
        for result in tick_results:
            pack.take_damage(result["damage"])
            pack_dot_accumulated += result["damage"]
            GameEvents.dot_ticked.emit("pack", result["type"], result["damage"], pack_dot_accumulated)
            any_dots_active = true
        # Emit dot_expired for any type that was active before tick but now has 0 stacks
        for dot_type in pre_tick_types:
            if pack.get_dot_count(dot_type) == 0:
                GameEvents.dot_expired.emit("pack", dot_type)
        if not pack.is_alive():
            _on_pack_killed()
            return

    # Process pack's DoTs on hero
    if hero.active_dots.size() > 0:
        # Snapshot active types BEFORE tick to detect expirations
        var hero_pre_tick_types := {}
        for dot_type in ["bleed", "poison", "burn"]:
            if hero.active_dots.any(func(d): return d["type"] == dot_type):
                hero_pre_tick_types[dot_type] = true

        var tick_results := hero.process_dot_tick()
        for result in tick_results:
            # Route through defense calculator for resistance + ES split
            var dot_result := DefenseCalculator.calculate_dot_damage_taken(
                result["damage"],
                result["element"],
                hero.get_total_fire_resistance(),
                hero.get_total_chaos_resistance(),
                hero.get_current_energy_shield()
            )
            hero.apply_damage(dot_result["life_damage"], dot_result["es_damage"])
            hero_dot_accumulated += dot_result["life_damage"] + dot_result["es_damage"]
            GameEvents.dot_ticked.emit("hero", result["type"], dot_result["life_damage"] + dot_result["es_damage"], hero_dot_accumulated)
            any_dots_active = true
        # Emit dot_expired for any hero DoT type that transitioned to 0 stacks
        for dot_type in hero_pre_tick_types:
            if not hero.active_dots.any(func(d): return d["type"] == dot_type):
                GameEvents.dot_expired.emit("hero", dot_type)
        if not hero.is_healthy():
            _on_hero_died()
            return

    # Stop timer if no active DoTs remain
    if not any_dots_active:
        dot_tick_timer.stop()
```
